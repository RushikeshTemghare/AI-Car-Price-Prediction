## 📌 Project Overview

This project was developed as part of my **MSc Data Science & Artificial Intelligence** 
coursework. It investigates the use of AI and machine learning techniques to accurately 
predict sports car prices based on various features such as brand, engine size, mileage, 
and year of manufacture.

Accurate pricing models are valuable for:
- 🏭 **Manufacturers** — competitive market positioning
- 🤝 **Dealers & Resellers** — optimising inventory and pricing
- 👤 **Buyers & Enthusiasts** — making informed purchase decisions

## 📊 Dataset

- **Source:** Car pricing historical data
- **Size:** 301 rows × 9 columns
- **Features:**

| Feature | Description |
|---|---|
| Car_Name | Name of the car |
| Year | Year of manufacture |
| Selling_Price | Price the car was sold at |
| Present_Price | Current ex-showroom price |
| Kms_Driven | Kilometres driven |
| Fuel_Type | Petrol / Diesel / CNG |
| Seller_Type | Dealer or Individual |
| Transmission | Manual or Automatic |
| Owner | Number of previous owners |

---

## 🤖 AI Approaches Used

| Model | Purpose |
|---|---|
| Linear Regression | Baseline prediction model |
| Decision Trees | Handling non-linear patterns |
| Random Forest | Ensemble — reducing overfitting |
| Gradient Boosting (GBM) | Improving accuracy iteratively |
| Neural Networks | Capturing complex relationships |

---

## 📈 Results

- ✅ **R² Score (Linear Regression):** 0.88
- ✅ Ensemble methods outperformed individual models
- ✅ Random Forest and GBM provided the most accurate predictions
